Rating

Depending on how the Project Manager set up the evaluation, you will be asked to rate one alternative with respect to one objective on each screen, or all alternatives with respect to one objective on each screen, or one alternative with respect to all objectives on each screen.

 

Rating all alternatives with respect to one objective

 

Each row shown above are the alternatives to be evaluated with respect to one objective: Vendor/Partner Access.

 

The yellow highlight indicates that AS/400 Replacements is being evaluated with respect to Vendor/Partner Access.

 

The fastest way to enter a rating is to click on an intensity name, such as "Good" at the right side of the page.

 

After clicking the intensity name, the highlight advances to the next alternative.

 

The priority corresponding to the verbal intensities are shown also at the right of the page (Good - 44.216%).

 

You can enter your judgment by clicking on the drop down menu and selecting an intensity for each row:

 

If you would like to enter a rating other than those corresponding to the intensities, you can enter a direct value between and including 0 and 1.

 

 

 

Click to enter comments. Comments with entered data are shown with darker shade. Hovering on the icons will display a tooltip to let you have a quick view of the comments entered.

Information Documents

The Project Manager can specify to show or hide information documents during the evaluation. Information document can be displayed as frames or tooltips.

 

When information documents are set to be displayed, the Project Manager can update them in their Evaluation page.

Navigation:

The easiest and most common way to proceed through the evaluation is sequential -- by clicking the button.   You can go back to a previous step using the button. If you have previously entered judgments and requested to go back to an earlier step (see below) you click to jump to the next step that is un-assessed.

 

Non Sequential Navigation:

 

If the project manager enables displaying the navigation box, you can click on any step to move either forward or backward.

 

Depending on options set by the project manager, you may have considerable flexibility in navigating through the evaluation without sequentially going step by step.  We recommend that you do this only if you have used Comparion before.  

 

The icon  at the bottom left of the screen can be used to display a pop up a view of the hierarchy.  By clicking on any element in the hierarchy, you will jump to the first screen that elicits judgments' with respect to' that element. The 'with respect to' for the current step is shown in blue.
